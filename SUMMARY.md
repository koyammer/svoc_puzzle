# SVOC パズル — アプリ詳細サマリー

## 概要

英文法の「文の要素（SVOCM）」を学習するための Flutter 製クイズアプリ。  
主語(S)・動詞(V)・目的語(O)・補語(C)・修飾語(M)の5つの役割を正しく識別する力を、ゲーム感覚でトレーニングできる。

## ゲームモード

### 1. 分解モード

英文の各単語（フレーズ）を S / V / O / C / M のスロットに分類するモード。

**UI の流れ：**
1. 画面上部に英文カード表示
2. 常に `S・V・O・O・C・M` の6スロットを固定表示（パターンのヒントを与えない）
3. 単語バンクに**シャッフルされた**トークンが並ぶ
4. 単語をタップ → スロットをタップして配置
5. 全トークンを配置したら「確認する」ボタンが出現
6. 一括判定：正しければ完了、間違いは赤くフラッシュして単語バンクに返却

**スコアリング：**
- ミスなし完答：10 × コンボ倍率 点
- ミスあり完答：10 点（コンボリセット）

**難易度設計のポイント：**
- 単語の順序がランダムにシャッフルされる（左から順に並べるだけでは解けない）
- 問題の出題順もランダム
- 6スロット固定表示により、文型が何かは自分で判断しなければならない
- 一括確認方式により「とりあえず置いてみる」試行錯誤を意識的に行う

---

### 2. 文型判断モード

英文を見て、文型（SV / SVC / SVO / SVOO / SVOC）を3択から選ぶモード。

**UI の流れ：**
1. 英文カードを表示
2. 3択のボタンから文型を選択
3. 正解：緑グラデーション、不正解：赤グラデーション にボタンが変化
4. 1.4秒後に次の問題へ自動遷移

**スコアリング：**
- 正解：10 × コンボ倍率 点

## レベル構成

各レベル32問のプールから、毎回ランダムに**5問**を出題。

| レベル | 内容 | 文型 | 問題数 |
|--------|------|------|--------|
| Level 1 | 基本文 | SV / SVC / SVO | 32問 |
| Level 2 | 発展文 | SVOO / SVOC | 32問 |
| Level 3 | 修飾語あり | SV〜SVOC + M | 32問 |

**合計：96問**

### 問題データの構造（`Question` クラス）

```dart
Question(
  id: 'q101',
  sentence: 'She gave me a book.',
  level: 2,
  pattern: 'SVOO',
  tokens: ['She', 'gave', 'me', 'a book'],   // 分割済みトークン
  answer: {'S': ['She'], 'V': ['gave'], 'O': ['me', 'a book']},
)
```

- `tokens`：画面に表示される単語チップの単位（複数語フレーズも1トークン扱い）
- `answer`：役割 → トークンのマッピング。Oが2つある場合は `'O': ['me', 'a book']` のようにリストで持つ
- `roleFor(token)`：トークンからその役割を逆引きするメソッド

## プレイ履歴機能

ゲーム終了時に結果を端末にローカル保存し、履歴画面で確認できる。

**保存データ（`GameResult` クラス）：**

| フィールド | 内容 |
|-----------|------|
| `id` | Unix ミリ秒タイムスタンプ |
| `playedAt` | プレイ日時（ISO 8601） |
| `mode` | `'decompose'` または `'pattern'` |
| `level` | 1 / 2 / 3 |
| `score` | 獲得スコア |
| `correct` | 正解問題数（分解：ミスなし完答数 / 文型：初回正解数） |
| `total` | 出題問題数（常に5） |

**保存方法：**
- `shared_preferences` に JSON 文字列のリストとして保存
- 最大200件保持（超過分は古い順に削除）

**履歴画面の表示：**
- サマリーカード：総プレイ数・全体正解率・最高スコア
- セッションリスト（新しい順）：日時・モード・レベル・正解数・スコア

## スコアリング詳細

### コンボシステム

連続してミスなし（分解）or 初回正解（文型判断）をすると倍率が上がる。

```
1連続：10点
2連続：20点
3連続：30点
n連続：10 × n 点
ミス：コンボリセット（10点固定）
```
